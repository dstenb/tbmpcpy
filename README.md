tbmpcpy
=======

TBMPCY is a terminal MPD client written in Python. The client tries to adhere
to the style of vi, i.e. simple yet effective key bindings and a command line.

Requirements
------------
TBMPCY is written for Python 2.7. It requires the following libraries:

* termbox
* python-mpd or python-mpd2

Help
----
A list of key bindings can be found in the program by pressing 'h'.
